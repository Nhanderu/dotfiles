# Nhanderu's dotfiles

My own dotfiles. Yay.

## Steps to glory

```sh
# Installs configurations and scripts.
DOTFILES=~/projects/nhanderu/dotfiles ./bootstrap.sh

# Installs packages.
install-pkgs
```
